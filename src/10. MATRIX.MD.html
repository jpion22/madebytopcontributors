## Description The `MATRIX` custom formula combines the [`ARRAYFORMULA`][1], [`INDEX`][2] and [`SORT`][4] formula into one, which makes it very versatile in returning a sorted and modified array (matrix). More information can be found on the [help][3] page of the `MATRIX` formula, which also has a screencastify video available. ## Syntax MATRIX(range[, opt_row][, opt_col][, opt_sort]) ## Usage - `MATRIX(A:C, {1,4,5}, "2", false)` Sorts the complete range (only column A and C + row 1,4 and 5) descendingly - `MATRIX(A1:C8, , , true)` Sorts the complete range ascendingly - `MATRIX(A1:C8, ,"3")` Outputs only column 1 and 2 - `MATRIX(ARRAYFORMULA(VALUE(A1:C8)), "3", , 1, false, 3, true)` Sorts the range first by column 1, descendingly, then column 3 ascendingly [1]: https://support.google.com/docs/answer/3093275 [2]: https://support.google.com/docs/answer/3098242 [4]: https://support.google.com/docs/answer/3093150 [3]: https://sites.google.com/view/madebytopcontributors/custom-formulas/matrix
