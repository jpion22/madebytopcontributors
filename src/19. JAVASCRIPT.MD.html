## Description We've added a few Java Script based formula. More information can be found on the [help][1] page of the `JAVASCRIPT` formulas, which also has a screencastify video available. - JS_MAP - JS_SORT If you want to have more Java Script goodness added, use the [Google Forms][1] on our website. ## Syntax and Usage JS_MAP(E1:F5;"i > 2 ? cell : cell + 1") Outputs a range that is incremented when the relative row is larger than 2.
