## Description
The ROWTOTAL function returns a one-column array of aggregations of each row in the referenced array. The type of aggregation is specified by the function_code argument, and is analogous to the function codes used in the native SUBTOTAL function:
1. AVERAGE 2. COUNT 3. COUNTA 4. MAX 5. MIN 6. PRODUCT 7. STDEV 8. STDEVP 9. SUM 10. VAR 11. VARP If you want to have suggestions for more additions, please use the [Google Forms][1] on our website. ## Syntax =ROWTOTAL(function_code, range[, header]) All of the aggregations used in SUBTOTAL are also supported in ROWTOTAL. More information can be found on the [help][2] page of the ROWTOTAL formula.
