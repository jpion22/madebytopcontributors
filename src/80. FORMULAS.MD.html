## Description Formulas have a common utilitarian behavior and usually do not go beyond a single cell - TEXT2CHARACTER - TEXT2REVERSE - TRANSLITERATE - RETRANSLITERATE If you want to have more Java Script goodness added, use the [Google Forms][1] on our website. ## Syntax and Usage TEXT2CHARACTER("formulas") > Presents the text as individual characters vertically. Use the TRANSPOSE formula to present them horizontally. [1]: https://sites.google.com/view/madebytopcontributors/custom-formulas/formulas
