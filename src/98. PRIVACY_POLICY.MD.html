## Add-on The Custom Formulas Add-on uses the class [SpreadsheetApp][1] to create a menu and the class [HtmlService][2] to show a context related help window of the formula selected. In this process, no user information is collected, stored or used otherwise. Furthermore, the UTILITIES formulas use the class [SpreadsheetApp][1] to retrieve specific user spreadsheet information. In this process, no user information is collected, stored or used otherwise. Authentication of the Add-on is solely based on this use-case and scope. ## Formulas The Custom Formulas themselves are constructed in such a way that they only use pure Google Apps Script JavaScript. Even services that do not require authentication are omitted, to avoid user data being stored elsewhere. [1]: https://developers.google.com/apps-script/reference/spreadsheet/spreadsheet-app [2]: https://developers.google.com/apps-script/reference/html/html-service
