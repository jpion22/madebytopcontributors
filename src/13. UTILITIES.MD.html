## Description
We've added all kinds of utility functions that can retrieve spreadsheet info, like:
- remaining cells in sheet - remaining cells in spreadsheet - cells remaining to spreadsheet limit of 2 million - spreadsheet name - sheet name - total number of sheets - sheet index - reverse text - count number of formulas used in either range or sheet - retrieve the formula text If you want to have more utilities added, please use the [Google Forms][1] on our website. ## Syntax and Usage =UTIL_CELLSREMAININGSHEET() Outputs the the remaining cells in the active sheet.
