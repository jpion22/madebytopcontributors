## Description
We've added BRAILLE and MORSE text convertors, array enabled:
- TEXT2BRAILLE - BRAILLE2TEXT - TEXT2MORSE - MORSE2TEXT As a reference to the Braille converter, we used [Unified English Braille (UEB)][1]. No shortcuts, combinations are implemented, but a single translation of each character. For the Morse converter we used the [International Morse code][2] published by the International Telecommunication Union in 2009. ## Syntax and Usage TEXT2MORSE(B27:B28) or TEXT2MORSE(B27:B28; "&") TEXT2BRAILLE(B35:B36) or TEXT2BRAILLE(B35:B36; " / ") Outputs the array of either Morse or Braille code. ## References - http://www.brailleauthority.org/ueb/symbols_list.pdf - https://www.itu.int/dms_pubrec/itu-r/rec/m/R-REC-M.1677-1-200910-I!!PDF-E.pdf [1]: http://www.brailleauthority.org/ueb/symbols_list.pdf [2]: https://www.itu.int/dms_pubrec/itu-r/rec/m/R-REC-M.1677-1-200910-I!!PDF-E.pdf
